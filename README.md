# このプログラムについて

このプログラムは、力士の検索を行うプログラムになります。
sekitoriコマンドにより、力士の検索を行うことが出来ます。

```sh
$ sekitori search [-a, -d] word
```

## インストール

Add this line to your application's Gemfile:

```ruby
gem 'sekitori-search'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install sekitori-search
